\subsection{Detailed Description}
General P\+ID implementation for A\+VR. 

Discrete P\+ID controller implementation. Set up by giving P/\+I/D terms to Init\+\_\+\+P\+I\+D(), and uses a struct \hyperlink{struct_p_i_d___d_a_t_a}{P\+I\+D\+\_\+\+D\+A\+TA} to store internal values.
